# Obsidian Vault (openclaw-homeassistant)

Este vault está preparado para capturar:
- ideas rápidas (`00-Inbox/`)
- avances por proyecto (`10-Projects/`)
- contenido marca personal (`20-Brand/`)
- bitácora diaria (`30-Daily/`)

Uso recomendado:
1. Abrir esta carpeta como Vault en Obsidian Desktop.
2. Ejecutar los scripts de `brand-automation/scripts/` para sincronizar borradores y reportes.
3. Commit frecuente para dejar trazabilidad del trabajo nocturno.
